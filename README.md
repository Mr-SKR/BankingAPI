# BankingAPI

Usage
-----

Set environment variables for SECRET_KEY, RDS_DB_NAME, RDS_USERNAME, RDS_PASSWORD, RDS_HOSTNAME, RDS_PORT that gets referenced in settings.py(connects to postgres by default. Change that if you want)


Runserver


Obtain username and password to use the banking API. Click here to [SignUp](http://127.0.0.1:8000)
  
To generate a JWT token using above obtained username and password, you can make the below curl request.

```

  curl -X POST -H "Content-Type: application/json" -d "{\"username\": \"<username>\", \"password\": \"<password>\"}" http://127.0.0.1:8000/api/token/
```

Example API calls:

> GET API to fetch a bank details, given branch IFSC code
```

curl -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX3BrIjoxLCJ0b2tlbl90eXBlIjoiYWNjZXNzIiwiY29sZF9zdHVmZiI6IuKYgyIsImV4cCI6MTIzNTY3LCJqdGkiOiJjNzE4ZTVkNjgzZWQ0NTQyYTU0NWJkM2VmMGI0ZGQ0ZSJ9.ekxRxgb9OKmHkfy-zs1Ro_xs1eMLXiR17dIDBVxeT-w" http://127.0.0.1:8000/api/branch/ABHY0065001
```
where 'ABHY0065001' is the IFSC code
> GET API to fetch all details of branches, given bank name and a city
```

curl -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX3BrIjoxLCJ0b2tlbl90eXBlIjoiYWNjZXNzIiwiY29sZF9zdHVmZiI6IuKYgyIsImV4cCI6MTIzNTY3LCJqdGkiOiJjNzE4ZTVkNjgzZWQ0NTQyYTU0NWJkM2VmMGI0ZGQ0ZSJ9.ekxRxgb9OKmHkfy-zs1Ro_xs1eMLXiR17dIDBVxeT-w" "http://127.0.0.1:8000/api/bank?name=hdfc+bank&city=bangalore"
```
Spaces must be replaced either with '+' or '%20'. Example: hdfc+bank
> All the above API calls support limit & offset parameters
```

curl -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX3BrIjoxLCJ0b2tlbl90eXBlIjoiYWNjZXNzIiwiY29sZF9zdHVmZiI6IuKYgyIsImV4cCI6MTIzNTY3LCJqdGkiOiJjNzE4ZTVkNjgzZWQ0NTQyYTU0NWJkM2VmMGI0ZGQ0ZSJ9.ekxRxgb9OKmHkfy-zs1Ro_xs1eMLXiR17dIDBVxeT-w" "http://127.0.0.1:8000/api/bank?name=hdfc+bank&city=bangalore&limit=10&offset=5"
```
Default values for limit(100) and offset(0) will be applied if not specified externally
